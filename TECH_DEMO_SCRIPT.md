# Palabam Tech Demo Script
**Duration:** 3-5 minutes  
**Format:** Screen recording with voiceover (OBS corner cam)

---

## 1. Introduction & Problem (30 seconds)

**Topic:** The Challenge

**Say:**
> "Middle school educators spend hours manually identifying vocabulary gaps in student writing. Palabam automates this using AI to analyze transcripts and recommend personalized vocabulary words tailored to each student's proficiency level."

**Action:** Show landing page, navigate to login

---

## 2. Story Spark - Input & Analysis (1.5-2 minutes)

**Topic:** Automated Gap Detection

**Say:**
> "Let me show you Story Spark. I'll input a student's writing sample—either by typing or using voice input."

**Action:** 
- Navigate to Story Spark page
- Select a student (if teacher view)
- Type or paste a sample transcript (e.g., "I went to the store yesterday. It was fun. I bought some stuff.")
- Click "Submit Story"

**Say:**
> "The system analyzes the transcript using NLP, compares it against COCA and Lexile corpora, and identifies vocabulary gaps."

**Action:** Wait for processing, show progress bar

**Say:**
> "Here are the results. The system calculated the student's vocabulary level and recommended 5-7 personalized words at their Zone of Proximal Development—words that are challenging but achievable."

**Action:** 
- Point to vocabulary level badge
- Scroll through recommended words
- Highlight: word, definition, example, difficulty score, relic type (whisper/echo/resonance/thunder)

---

## 3. Educator Dashboard (1-1.5 minutes)

**Topic:** Centralized Recommendations

**Say:**
> "Now let's see the educator dashboard where all student recommendations are centralized."

**Action:** Navigate to dashboard

**Say:**
> "Here's the overview: total students, analyzed students, words recommended, and classes. The dashboard shows all my students with their personalized vocabulary recommendations."

**Action:**
- Point to stats cards
- Scroll through student list
- Click on a student to show their recommended words
- Show filter/search functionality

**Say:**
> "Each student has their own vocabulary profile. I can see which words were recommended when, track progress over time, and view transcript history."

**Action:** Show student detail view with recommendations

---

## 4. Export & Value Proposition (30 seconds)

**Topic:** Export & Workflow Integration

**Say:**
> "I can export recommendations as CSV to share with parents or include in lesson plans."

**Action:** Click export button (if available), show CSV download

**Say:**
> "Palabam reduces manual analysis time by 50% while providing evidence-based vocabulary recommendations that accelerate student learning. That's Palabam—automated vocabulary gap detection for middle school educators."

**Action:** Return to dashboard, show final overview

---

## Key Talking Points (Reference)

- **Automated Gap Detection:** AI analyzes using NLP (spaCy/NLTK) + COCA/Lexile corpora
- **Personalized Recommendations:** 5-7 ZPD-balanced words (70-80% learnability)
- **Educator Dashboard:** Centralized view of all students and recommendations
- **Export Capabilities:** CSV/PDF for lesson planning and parent sharing
- **Time Savings:** 50% reduction in manual analysis time
- **Pedagogical Foundation:** Evidence-based (ZPD, explicit word selection)

---

## Demo Flow Checklist

- [ ] Login as teacher
- [ ] Navigate to Story Spark
- [ ] Select student
- [ ] Input transcript (type or voice)
- [ ] Show analysis results (vocabulary level + recommendations)
- [ ] Navigate to dashboard
- [ ] Show stats overview
- [ ] Show student list with recommendations
- [ ] Click student detail view
- [ ] Show export functionality
- [ ] Closing statement

---

## Tips for Recording

- **Pace:** Speak clearly, pause briefly after each action
- **Highlighting:** Use cursor to point at key UI elements
- **Transitions:** Smooth navigation between pages
- **Errors:** If something fails, acknowledge briefly and continue
- **Timing:** Aim for 3.5-4 minutes total
